# AI 语义匹配工具

## 简介

AI 语义匹配工具是一款基于阿里云 DashScope API 的图形用户界面 (GUI) 应用程序，旨在对两个 Excel 文件中的数据进行语义匹配。该工具支持一对多匹配，并保留每条数据的前 3 条高分结果。

## 功能

- **API 配置**：
  - 输入并保存 API Key。
  - 校验 API Key 的有效性。
  - 显示 API 状态（红灯无效，绿灯有效）。
- **文件选择与配置**：
  - 支持选择 A 表（待匹配）和 B 表（存货档案）。
  - 配置大语言模型的温度值（控制生成随机性）。
  - 显示向量化状态（蓝灯表示完成）。
- **语义匹配**：
  - 使用阿里云的文本嵌入 API 生成向量。
  - 使用大语言模型 API 计算语义相似度。
  - 支持多线程并发处理，提高匹配效率。
  - 点击“开始匹配”按钮后才开始向量化和匹配。
  - 支持匹配过程中随时停止，停止后仅输出已完成的数据。
- **结果输出**：
  - 匹配结果保存为新的 Excel 文件，包含匹配内容、得分和状态。
  - 匹配完成后弹窗提示“匹配全部完成”。
- **运行日志**：
  - 实时显示操作日志，记录匹配过程中的状态和错误信息。


## 业务逻辑


### 大体流程
1. A表文本向量化（API调用）
          ↓
2. Cosine相似度计算，找出Top20（数值计算，极快）
          ↓
3. 把Top20的B表数据发给LLM：
   "从这20条中找出与A最相似的1条，输出索引"
          ↓
4. LLM返回最终匹配结果

## 使用方法

1. **安装依赖**：
   确保已安装以下 Python 库：

   ```bash
   pip install requests numpy openpyxl
   ```

2. **运行程序**：
   双击运行 `main.py`，或在终端中执行：

   ```bash
   python main.py
   ```

3. **配置 API Key**：
   - 在程序界面中输入阿里云 DashScope 的 API Key。
   - 点击“校验 API”按钮，确保 Key 有效。

4. **选择文件**：
   - 选择 A 表（待匹配）和 B 表（存货档案）。
   - 配置大语言模型的温度值。

5. **开始匹配**：
   - 点击“开始匹配”按钮。
   - 等待程序完成匹配，匹配结果将保存为新的 Excel 文件。
   - 匹配完成后，点击“打开结果文件”按钮查看结果。

6. **停止匹配**：
   - 在匹配过程中，可随时点击“停止运行”按钮。
   - 停止后仅输出已完成的数据。

## 文件说明

- `main.py`：程序主文件，包含所有功能逻辑。
- `config.json`：保存用户的 API Key 配置。
- `vector_cache/`：存储生成的向量缓存文件，避免重复计算。
- `run_log.log`：运行日志文件，记录操作状态和错误信息。

## 注意事项

- **API Key**：
  - 请确保使用有效的阿里云 DashScope API Key。
- **Excel 文件格式**：
  - A 表和 B 表需为 `.xlsx` 格式。
  - 数据需从第 2 行开始，第 1 行作为表头。
- **向量缓存**：
  - 如果 B 表内容发生变化，请删除 `vector_cache` 目录下对应的缓存文件，确保重新生成向量。
